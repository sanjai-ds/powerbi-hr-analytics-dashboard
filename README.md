# Power BI HR Analytics Dashboard

##  Project Overview

This project showcases an **end-to-end Power BI dashboard** built using an Excel dataset.
The goal is to transform raw data into meaningful insights that support better business decision-making.

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel (Data Source & Cleaning)
* Power BI (Dashboard Development)
* Power Query (Data Transformation)
* DAX (Measures & KPIs)

---

## 📌 Key Features

### 🔹 Data Preparation
* Cleaned and transformed raw data using Excel and Power Query
* Handled missing values and ensured data consistency

### 🔹 Data Modeling

* Created relationships between multiple tables
* Structured data model for accurate reporting

### 🔹 Visualization Design

* Built interactive dashboards including:

  * Team-wise analysis
  * Manager-wise performance
  * Gender distribution
  * Job level insights

### 🔹 Interactivity

* Added:

  * Filters & slicers
  * Page navigation
  * Bookmarks for better user experience

### 🔹 Data Storytelling

* Designed a clean and structured dashboard
* Converted raw data into actionable insights

---

## 📊 Dashboard Insights

* Gender diversity analysis

* Performance trends and patterns
